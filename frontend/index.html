Migra LXC
v1.0.0 · Proxmox Migration Tool
Strumento web-based per migrare container LXC tra nodi Proxmox non in cluster, tramite uno storage bridge condiviso (NFS/CIFS).
Flusso operativo
01 Stop container sorgente
02 Backup su bridge storage
03 Restore su host destinazione
04 Avvio container (opzionale)
05 Cleanup sorgente e backup
